# LCMD App

## Build
```
lzc-cli project build -o you-awesome.lpk
```
a lpk package will be built in the current directory.

## Install
```
lzc-cli app install you-awesome.lpk
```

## Develop
```
lzc-cli project devshell -b
```

## become a lcmd application developer
register a developer account at https://developer.lazycat.cloud. After registering the developer account, you can apply for your own application and publish it to [Store](https://appstore.lazycat.cloud).

## use lzc-cli to publish application
after you apply for an application successfully, subsequent application upgrades can be released directly through 
`lzc-cli appstore publish <lpk path>`

# view published apps
After successful publishing, you can view your app at https://appstore.lazycat.cloud.
